Please use a browser that supports "canvas"

For drawing a picture, I made a butterfly. Click on the "Draw a Butterfly" button to render the picture.
For the awesomness part, I implemented 2 things:
- Paint transparency: We can use the transparency slider to make the color of the shape transparent.
- Pizza shaped paint brush: Click on the Button "Pizza paint brush" to use it. This brush draws pizzas!
Note: We can make all the other shapes transparent (Circle, Square, Triangle) except the pizzas.
